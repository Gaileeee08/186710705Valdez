Introduction

TechScript is a powerful, easy-to-learn programming language designed for both beginners and experienced developers. It focuses on simplicity and efficiency, making it ideal for rapid development.

Setup

Follow these steps to set up TechScript on your system:

  1. Download the TechScript installer from the official website.
  2. Run the installer and follow the on-screen instructions.
  3. Set up environment variables as necessary.
  4. Verify the installation by running techscript --version in your terminal.

Syntax

TechScript syntax is designed to be straightforward and easy to understand. Here are some key syntax rules:

  • Variables are declared using let or const.
  • Functions are defined using the function keyword.
  • Comments are written with // for single-line comments and /* ... */ for multi-line comments.

Functions

Functions in TechScript are blocks of code designed to perform a particular task. Here's the basic syntax:


function greet(name) {
    return "Hello, " + name + "!";
}

You can also use arrow functions for a more concise syntax:


const greet = (name) => "Hello, " + name + "!";

Examples

Here are some examples to get you started with TechScript:


// Basic Hello World
console.log("Hello, World!");

// Simple function
function add(a, b) {
    return a + b;
}
console.log(add(5, 3)); // Outputs: 8
